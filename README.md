# Logistics Data Processing Automation Tool

## Overview
This project automates a specific, repetitive data processing task within a logistics operation.

## Problem
A recurring operational task required manual sorting and structuring of pre-defined data from a standardised file format. This process was time-consuming and prone to human error.

## Solution
Developed a targeted Excel VBA macro to:
- Automatically process and sort structured input data
- Ensure consistency across repeated operational workflows
- Eliminate the need for manual intervention

## Scope
This tool is designed for a specific dataset structure and operational use case, reflecting a real-world business process rather than a generic solution.

## Tools Used
- Microsoft Excel
- VBA (Visual Basic for Applications)

## Impact
- Reduced manual processing time for a recurring task
- Improved accuracy and consistency of operational data
- Streamlined a key part of the workflow

## Future Improvements
- Expand to handle variable data formats
- Integrate with broader data processing pipelines
